**Fitness app**

This project is a terminal-based fitness application built in Java Visual Studio code. 
The goal of the app is to help users manage their fitness routine directly from the command line.

**Screenshot(s)**
<img width="195" height="179" alt="image" src="https://github.com/user-attachments/assets/4c5607b5-b899-4743-81e4-a48ac2c6bbda" />



The applicaion allows users to:
- Enter and manage personal fitness data.
- View and select different exercises.
- Generate and follow a daily schedule.
- Generate and follow a weekly schedule.
- Work with a varied exercises to keep workouts diverse.

**Features**
- User data input.
- Exercises.
- Daily & weekly schedule.

**Run**
- javac Fitness.java
- java Main
